## 4) UI to Python IPC and input shape control

```mermaid
sequenceDiagram
  participant JS as WebView_JS
  participant Py as desktop_visualizer_py
  participant GTK as GTK_Window
  participant Cairo as cairo_Region

  JS->>Py: set document title with JSON action menu toggle or menu resize or close
  Py->>Py: on title changed parses JSON
  alt menu toggle or menu resize
    Py->>GTK: resize window to target height
    Py->>GTK: set size request
    GTK->>Py: size allocate event
    Py->>Cairo: build input region gear rect and menu rect if open
    Py->>GTK: window input shape combine region
  else close
    Py->>Py: quit main loop
  end
```

## 5) auto_route_audio daemon

```mermaid
flowchart LR
  Start([start])
  Loop[loop until timeout about 20 seconds every 0.5s]
  RunP[call pactl list source outputs]
  Parse[parse output for WebKitWebProcess source outputs]
  Found{Found WebKit stream}
  Move[call pactl move source output id to target device]
  Success([exit thread])
  Sleep[sleep 0.5s]
  Timeout([stop no routing found])

  Start --> Loop
  Loop --> RunP --> Parse --> Found
  Found -- yes --> Move --> Success
  Found -- no --> Sleep --> Loop
  Loop -->|timeout| Timeout
```
